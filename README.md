🌦️ Weather App

A simple weather app that shows current weather details using your location or a searched city.

🚀 Features
📍 Get weather using your current location
🔍 Search weather by city name
🌡️ Shows temperature, wind speed, direction, and time
🗺️ Displays location on a map
🛠️ Technologies Used
HTML
CSS
JavaScript
Open-Meteo API
Leaflet.js (for map)
📂 Files
index.html – Main page
style.css – Design & layout
script.js – Weather logic & API calls
▶️ How to Run
Download or clone the project
Open index.html in your browser
⚠️ Note
Allow location access for automatic weather
Internet is required
